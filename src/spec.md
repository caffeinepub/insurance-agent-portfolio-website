# Specification

## Summary
**Goal:** Build a professional insurance agent portfolio website with emotional storytelling, dark luxury design, and lead capture functionality.

**Planned changes:**
- Implement hero section with dual CTAs, professional portrait, and animated gradient lines on dark luxury background
- Create trust/authority bar with stats and certification badges
- Build About section with emotional storytelling and mission-driven narrative
- Create five premium animated service cards (Life, Health, Term, Investment, Retirement)
- Build Why Choose Me section with seven key differentiators
- Create testimonials section with 3-5 emotional client stories
- Implement emotional close section with scarcity messaging and strong CTA
- Build contact section with premium form, WhatsApp button, sticky floating CTA, and calendar booking placeholder
- Apply dark luxury theme with navy/black background, gold (#D4AF37) and electric blue (#2563EB) accents
- Implement glassmorphism cards with smooth animations, parallax effects, and micro-interactions
- Add sticky header with smooth scroll navigation and floating WhatsApp icon
- Ensure mobile-first responsive design
- Apply SEO optimization with semantic HTML5 and proper heading hierarchy
- Implement backend form submission handler for lead capture

**User-visible outcome:** A fully functional, emotionally compelling insurance agent portfolio website with dark luxury aesthetics, smooth animations, lead capture form, and mobile-responsive design that visitors can browse and use to book consultations.
